# SimplicityPress

SimplicityPress is a simple, library-first static site generator focused on posts and pages.

## Features (planned)

- Library-first core usable from CLI and GUI
- TOML-based site configuration
- Markdown content with TOML front matter
- Jinja2 templates for flexible theming
- Clean `src/` layout with modern packaging

## Usage

After installing in editable mode:

```bash
python -m pip install -e .
```

You can access the CLI:

```bash
simplicitypress --help

simplicitypress build
```
